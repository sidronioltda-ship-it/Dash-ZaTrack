# Dash-ZaTrack

Dashboard premium criado com Next.js 15, TypeScript, Tailwind CSS v4 e componentes no padrao shadcn/ui.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- shadcn/ui + Radix primitives
- Recharts
- Lucide Icons

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra `http://localhost:3000` para visualizar o dashboard.
